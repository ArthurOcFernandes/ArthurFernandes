Meu primeiro e mais marcante contato com programação, Java é definitivamente a linguagem pela qual tenho mais afinidade e tempo dedicado; foi onde aprendi os conceitos de Orientação a Objetos e aprimorei meus conhecimentos em lógica computacional.

A linguagem que estudei na faculdade na matéria de Algorítmos e Lógica de Programação, foi fundamental para meu entendimento de como um computador funciona. Diria, sem hesitar que é um must para todo programador iniciante.

Desde que comecei a programar, já comecei a utilizar Linux por todas as facilidades que o sistema traz para programadores; aprendi a usar a CLI, e o básico de Shell Scripting, hoje utilizo o SO a maior parte do tempo.

MySql foi o banco de dados que decidi adotar como minha entrada ao mundo SQL. Conheço ate o momento o suficiente dos comandos DDL, DML e DQL para criar, manipular e realizar consultas em bancos de dados simples.

Em javascript, aprendi a dinamizar sites simples manipulando o DOM e um pouco de JS para back-end. Ainda assim, tratando-se de back-end uso Java sempre que posso.

Minha experiencia com HTML e CSS se dá em construção de landing pages para trabalho da faculdade, bem como a construção desse personal portfolio. Nesta página, utilizei HTML5, CSS3 e Javascript para montar minha apresentação.
